# architecture Vjepa2VitlFpc64256

> V-JEPA 2 video world model — joint-embedding predictive architecture over video clips. A 24-block ViT encoder (1024d, 16 heads) feeds a 12-block predictor (384d, 12 heads) that forecasts target latents in the encoder's embedding space. Mask tokens, the EMA target encoder, and the latent regularizer are captured as verification rules (see below). Generated by `n-orca hf convert` from a Hugging Face config.

## verification rules
- latent-dim-consistency: encoder_latents and predicted_latents share the embedding dim d_model — the predictor forecasts in the encoder's representation space, never in pixel space.
- joint-embedding: the training loss compares predicted_latents against an EMA target-encoder's latents (L1/L2 in feature space).
- stop-gradient: the target encoder is an EMA copy of the context encoder; gradients do not flow into the target branch (not modeled as a forward node here).
- mask-tokens: the predictor queries 10 learnable mask tokens for masked/target positions (pred_num_mask_tokens); modeled here as in-place latent forecasting over the N context tokens.
- positions: encoder/predictor attention uses 3D rotary position embeddings (RoPE), applied inside attention — left implicit here.
