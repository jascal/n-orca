# architecture LewmPusht

> LeWorldModel (LeWM) action-conditioned world model — joint-embedding predictive architecture over images. A 12-block ViT encoder (192d, 3 heads) feeds a 6-block predictor (192d, 16 heads) that forecasts target latents in the encoder's embedding space. The predictor is action-conditioned (action_dim=10) with a 2048-wide projector head. Mask tokens, the EMA target encoder, and the latent regularizer are captured as verification rules (see below). Generated by `n-orca hf convert` from a Hugging Face config.

- latent-dim-consistency: encoder_latents and predicted_latents share the embedding dim d_model — the predictor forecasts in the encoder's representation space, never in pixel space.
- joint-embedding: the training loss compares predicted_latents against an EMA target-encoder's latents (L1/L2 in feature space).
- stop-gradient: the target encoder is an EMA copy of the context encoder; gradients do not flow into the target branch (not modeled as a forward node here).
- action-conditioning: the predictor is conditioned on an embedded action added to the context tokens; the per-step action embedding is broadcast across the N token axis upstream.
- prediction-horizon: the predictor rolls the latent state forward over 3 frame(s) of control (predictor.num_frames).
- sigreg-regularizer: a LeJEPA SIGReg (sketched isotropic-Gaussian) penalty regularizes the latent distribution to prevent collapse; it is a loss term, not part of the forward graph rendered here.
- positions: fixed 2D sin-cos position embeddings are added to the patch tokens — left implicit here (parameter-free).
