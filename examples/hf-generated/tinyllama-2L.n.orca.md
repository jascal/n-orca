# architecture TinyllamaStyle

> Llama-family decoder-only transformer. 2 pre-norm blocks with RMSNorm (approximated as LayerNorm), rotary attention, and SwiGLU feed-forward (approximated as FeedForward). Generated by `n-orca hf convert` from a Hugging Face config.

## hyperparameters

| Name       | Type  | Default |
|------------|-------|---------|
| vocab_size | int   | 1000    |
| d_model    | int   | 128     |
| n_layer    | int   | 2       |
| n_heads    | int   | 4       |
| d_ff       | int   | 256     |
| max_pos    | int   | 256     |
| dropout    | float | 0.0     |

## tensors

| Name      | Shape              | Dtype   |
|-----------|--------------------|---------|
| input_ids | (B, S)             | int64   |
| logits    | (B, S, vocab_size) | float32 |

## layer input_ids [input]
> Token IDs

## layer tok_embed
> Token embeddings (rotary positions applied inside attention)
- op: Embedding(vocab_size, d_model)

## layer ln_1_0
- op: LayerNorm(d_model)

## layer attn_0
- op: MultiHeadAttention(d_model, n_heads, dropout)

## layer add_attn_0
- op: Add()

## layer ln_2_0
- op: LayerNorm(d_model)

## layer ffn_0
- op: FeedForward(d_model, d_ff, dropout)

## layer add_ffn_0
- op: Add()

## layer ln_1_1
- op: LayerNorm(d_model)

## layer attn_1
- op: MultiHeadAttention(d_model, n_heads, dropout)

## layer add_attn_1
- op: Add()

## layer ln_2_1
- op: LayerNorm(d_model)

## layer ffn_1
- op: FeedForward(d_model, d_ff, dropout)

## layer add_ffn_1
- op: Add()

## layer final_norm
> Final RMSNorm
- op: LayerNorm(d_model)

## layer lm_head
- op: Linear(d_model, vocab_size)

## layer logits [output]

## flow

|------------|------------|------------|

| add_attn_0 | ln_2_0     | r_0        |
| ln_2_0     | ffn_0      | r_0_n2     |
